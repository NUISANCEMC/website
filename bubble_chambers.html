Bubble chamber data

Over the years we've collected a fair amount of data from the community, and host a tarball of bubble chamber results here.

We do not claim this is a complete list, but it's somewhere to start if you're looking for bubble chamber data. The long term plan is to upload it to HEPData, which is an ongoing process.

Please let us know if you have any questions, and don't hesitate to let us know if this has been useful!

We also have a wiki page on various neutrino fluxes which tries to clarify which neutrino flux should be use with which experiment.
