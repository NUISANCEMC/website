Monte Carlo Event Files

Loading multi-generator events into NUISANCE is straight-forward, but it can time consuming to create the multiple generator event files needed for a single comparison.

Unfortunately NUISANCE doesn't handle event generation (yet!), but as a consolation we publicly host all MC files used to make our analyses.


We currently only provide the raw generator output format which is to be fed to NUISANCE, which requires the generator to be installed on the system.

We soon hope to provide official flat-trees (produced with nuisflat), allowing users do generator-vs-generator comparisons using TTree->Draw(), without requiring generators to be installed.
